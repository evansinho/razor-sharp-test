# Series Tracker

A simple app for adding two hexidecimal values.

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Built with

- React

### Prerequisites

- Node
- Npm

### Setup

- git clone `https://github.com/evansinho/razor-sharp-test`
- cd razor-sharp-test
- run npm install
- run npm run start
